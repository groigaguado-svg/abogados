# Lezama & Ferrer Abogados

![Lezama & Ferrer Abogados](./logo-lockup-optim.jpg)

Sitio web corporativo de **Lezama & Ferrer Abogados**, despacho independiente en Madrid y Barcelona especializado en derecho mercantil, laboral, civil, penal, fiscal y extranjería.

- **Producción:** [https://abogados-two-beta.vercel.app](https://abogados-two-beta.vercel.app)
- **Stack:** HTML + CSS + JS estático (sin build), desplegado en Vercel

## Estructura

- `index.html` — página completa (single-file: contenido, secciones legales en diálogos y estilos/imágenes embebidos en base64)
- `logo-lockup-optim.jpg`, `logo-monogram-optim.jpg` — identidad visual
- `facade-optim.jpg`, `portada-optim.jpg` — fotografía de sede y hero
